Приложение разработано на языке C# .NET Framework. База данных на MySql. Визуальная часть на XAML через WPF. Приложение состоит из 2 частей: администраторская и клиентская версия. Представляет собой классический магазин. Реализовано через паттерн MVVM с использование Entity Framework и таких паттернов проектирования как Command, Repository, Unit of Work...
